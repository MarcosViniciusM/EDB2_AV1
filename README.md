Importante!

Código aceita instruções no terminal do seguinte modo:

./timer <tamanho da entrada> <função a ser utilizada>

com o tamanho da entrada aceitando quaisquer int acima de 0, e função sendo 1 para Heapsort, e 2 para Selection Sort.

Caso não seja mandado nenhum valor no terminal, ele irá para o default, que é n = 32, e heapsort.
